# アップリンク社労士事務所 — サンプルアプリ集

架空の事務所想定で、10領域の **Googleスプレッドシート／GAS向けサンプル** を置いています。  
ヒアリング時に「こういう画面・表になります」と見せるための種です。

## 仮想事務所

詳細: [`_shared/OFFICE.md`](_shared/OFFICE.md)

- 事務所名: **アップリンク社労士事務所**
- 顧問先例: 株式会社サンプル製造（製造業・従業員42名）
- 使い方想定: **顧客のGoogle上にコピーして共作**（こちらのテナントにデータを集めない）

## Googleシートへの入れ方（共通）

1. 顧客（またはデモ用）Googleでスプレッドシートを新規作成  
2. 各フォルダの `data/*.csv` をタブとしてインポート（またはコピペ）  
3. `拡張機能 > Apps Script` に `Code.gs` を貼り付けて保存  
4. シートを再読み込みし、カスタムメニューから実行  

**顧客へこちらから入れる場合:** [`DEPLOY.md`](DEPLOY.md)（Cursor / clasp）

デモ用はこちら所有のGoogleでも可。**本番は顧客Googleをオーナーにする。**
